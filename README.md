# Anonymous Code Repository

This repository contains the code used to reproduce the experimental results reported in the accompanying anonymous submission.

## How to Run

To reproduce the main results, please use the provided shell scripts in the `scripts/` directory. For example:

```bash
sh scripts/cross_sectional.sh
sh scripts/overtime.sh
```

Please refer to the script names for different experiments. All dependencies are listed in the code or can be installed as needed (e.g., torch, numpy).
